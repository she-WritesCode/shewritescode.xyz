Here's how to structure each **Selected Work** entry as its own whiteboard "scene":

---

**Each project = a whiteboard cluster**, with:

- A **large sticky note** as the project title/headline (bold, slightly rotated, colour-coded per project)
- A **"problem" note** — small, scribbled-feeling — _"Estate management was manual. WhatsApp groups. Paper receipts."_
- A **"solution" note** — slightly larger, a different colour — what you built
- A **result tag** — like a label sticker or pin — _"₦500k/month · Zero errors"_
- **Tech stack** written like handwritten labels or tape-flagged chips along the bottom
- A **faint connector line or arrow** between the problem → solution notes, like you drew it on the board

---

**Colour-code each project** so they each feel distinct on the board:

| Project             | Sticky Colour          |
| ------------------- | ---------------------- |
| Alajo               | Yellow + green accent  |
| Veesitor            | Blue + pink accent     |
| AgricTrail          | Green + orange accent  |
| Prodeegi            | Purple + yellow accent |
| Noja360             | Coral + white accent   |
| Virtual Soul Clinic | Lavender + teal accent |

---

**Section transitions** between projects could look like:

- A horizontal "tape" divider drawn across the board
- A pinned index card with the project number (_01, 02, 03..._)
- Subtle board texture shift (slightly different paper/cork tone per zone)

---

**The hero** stays close to what you have — photo, name, sticky note taglines — but swap the current taglines to reflect your new positioning (_"I build the infrastructure behind products people rely on"_).

**The About section** becomes one cluster of sticky notes on the board — personal facts as individual notes, pinned together.
